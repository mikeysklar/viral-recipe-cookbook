#airfryer #lowcarb #quiche #recipe 

```
+------------------+-----------------------------------------------+
| Category         | Details                                       |
+------------------+-----------------------------------------------+
| Eggs             | 6 large eggs                                  |
| Broccoli         | 1 cup chopped, blanched 2 min, dried          |
| Ham              | Chicken sausage                               |
| Tomatoes         | 1 dozen cherry, sliced                        |
| Cheese           | 1/4 cups shaved asiago shredded               |
| Everything Bagel | 1 Tbl                                         |
| Prep             | Parchment paper square                        |
| Mix              | Blend eggs + seasoning                        |
| Add-ins          | Pour over broccoli + ham + most cheese        |
| Fill             | Pour pan, top w/ remaining cheese             |
| Air Fryer        | 350°F for 20 min                              |
| Done             | Lift tray, place on rack                      |
| Rest             | 10 min before slicing                         |
+------------------+-----------------------------------------------+
```
